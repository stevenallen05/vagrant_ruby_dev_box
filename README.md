### My Vagrant/Ansible dev box setup

Requires:
 * Vagrant
 * Ansible

Provides:
 * Postgresql
 * Redis
 * RabbitMQ
 * rbenv w/ ruby-build default-gems plugins
 * nodenv w/ node-build plugin
 * DNS-over-TLS to cloudflare
 * Nordvpn
 * Krypt.co for SSH key management
 * Unison sync between `~/host_home/gits` and `~/gits`
 * Mailhog
 * Ngrok
